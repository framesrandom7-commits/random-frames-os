# Leads Module

---

# Purpose

The Leads module manages every potential customer from the first contact until they either become a client or are marked as closed.

This module is the starting point of the entire Random Frames business workflow.

No project should exist without first originating from a Lead.

---

# Objectives

- Never lose a lead.
- Track every interaction.
- Schedule follow-ups.
- Measure conversion rates.
- Convert leads into clients with one click.
