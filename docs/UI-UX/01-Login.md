# Login Screen

---

# Purpose

The Login screen provides secure access to Random Frames OS.

It should be simple, modern, fast, and aligned with the Random Frames brand identity.

---

# Layout

Split Screen Layout

Left Panel (40%)

- Random Frames Logo
- Brand Tagline
- Background Illustration
- Version Number

Right Panel (60%)

- Login Form
- Remember Me
- Forgot Password
- Sign In Button

---

# Fields

Email Address

Password

Remember Me

Forgot Password

Login Button

---

# Validation

Email is required.

Password is required.

Display validation errors beneath each field.

---

# Buttons

Primary

- Sign In

Secondary

- Forgot Password

---

# Login Flow

User enters credentials

↓

Validate inputs

↓

Authenticate

↓

Success

↓

Dashboard

Failure

↓

Display error

Remain on Login page

---

# Error Messages

Invalid Email

Incorrect Password

Network Error

Server Error

Session Expired

---

# Security

Passwords must be masked.

HTTPS only.

Secure Session.

Auto Logout after inactivity.

Future Support:

- Google Sign In
- Two Factor Authentication
