# Dashboard

---

# Purpose

The Dashboard is the command center of Random Frames OS.

It provides a real-time overview of business performance, pending work, finances, upcoming deadlines, and quick access to every major module.

The Dashboard should allow the owner to understand the health of the business within a few seconds.
