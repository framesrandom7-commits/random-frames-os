# Clients Module

---

# Purpose

The Clients module is the central repository for all confirmed customers of Random Frames.

Every client profile acts as a 360-degree business dashboard containing contact information, communication history, projects, quotations, invoices, payments, documents, reminders, and business insights.

---

# Objectives

- Maintain a complete client history.
- Eliminate duplicate client information.
- Track every project.
- Improve repeat business.
- Monitor lifetime client value.
